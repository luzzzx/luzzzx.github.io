Проектирование

Для расчета проекта отправьте нам ваш рисунок планировки. На основании данных мы подготовим для вас проект деревянных конструкций и архитектурных решений (КД, АР). Мы работаем с профессиональными программа для деревянного домостроения "К-3 Коттедж", "SEMA", "Cadwork".

Что необходимо для расчета проекта?

Для расчета проекта отправьте нам ваш рисунок планировки. На основании данных мы подготовим для вас проект деревянных конструкций и архитектурных решений (КД, АР)/

Состав проекта:

  • Ведомость материалов из программы( стеновой материал, материалы перекрытий, доски на стропила и т.д.которые заложены в проекте, Винтовые опоры, шпильки, полотно джутовое)

  • 3д виды ( фасады, разрезы, планы этажей, планы балок (пол,потолок), план кровли, план стропил, развёртки скатов, развёртки стен, таблица проёмов, таблица пазов и торцов брёвен, таблица брёвен, таблица раскроя брёвен

  • По необходимости: планы по венцам, ведомость по венцам

  • Машинный файл в формате .bvn

Контроль качества выпускаемой продукции:

Первая ступень

Первоклассное сырье. Только из здоровой крепкой отборной северной древесины возможно создание надежного строительного материала для наших домов. Весь материал приобретается у крупных надежных лесозаготовителей Ямала и Севера Югры и проверенный на происхождение по единой системе ЛЕСЕГАИС.

Вторая ступень

Оптимальный процесс сушки для достижения максимальной крепости и однородности древесины. Управление процессом сушки осуществляется посредством постоянного контроля над влажностью доски и температурой в камере посредством датчиков, расположенных непосредственно на древесине.

Третья ступень

Полностью автоматизированная линия, связывающая в единую цепочку этап оптимизации и удаления дефектов материале, строгание и склейку доски в ламели. В итого достигается однородностьи максимальный уровень качества древесины в ламели. Для склеивания применяется один из самых современных и экологически чистых клеевых составов.

Четвертая ступень

Регулярные тесты в лаборатории выборочных образцов из каждой партии выпускаемой продукции на соответствие европейским стандартам качества и технологическим требованиям производимой продукции.
